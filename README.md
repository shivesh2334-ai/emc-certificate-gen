# EMC Medical Certificate Generator

An automated tool for 'Easy My Care' to generate professional medical certificates using OCR (Optical Character Recognition).

## Features
- **Prescription OCR:** Upload an image of a prescription to automatically extract text.
- **Manual Overrides:** Manually enter name, diagnosis, symptoms, and dates.
- **Letterhead Integration:** Renders text directly onto the EMC official letterhead.
- **Export Options:** Download as PDF, Share to WhatsApp, Email, or Copy text.

## Setup
1. Clone the repo
2. Run `npm install`
3. Place your `letterhead.png` inside the `public/` folder.
4. Run `npm start`
